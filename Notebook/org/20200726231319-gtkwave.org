#+title: gtkwave
Make sure you copy the /usr/share/gtkwave/examples/gtkwaverc file to your home directory (as .gtkwaverc) or to your VCD project directory.
It contains the prefs for a good configuration that most people find ergonomic.  It is not strictly necessary however.
