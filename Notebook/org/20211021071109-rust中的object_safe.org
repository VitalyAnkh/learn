#+title: Rust中的object safe
Object safe这个概念是为了限制可以做成trait object的trait.
一个trait是如此的才算object safe:
1. 类型绑定没有 ~Sized~ ，因为trait声明都有一个隐式的bound: ~Sized~ 。
2. 所有method都是object safe的。
一个method是object safe的，如果满足：
1. 类型绑定没有 ~?Sized~ ；
2. 方法签名的receiver只有 ~self~ 。

具体查看Rust的rfc: https://rust-lang.github.io/rfcs/0255-object-safety.html
